# Naf CLI 
A Node API with Fastify Generating project.

## Prerequisites
- Nodejs
- MongoDB

## Setting up

#### Installation
`npm install -g @vixnguyen/naf`

#### Project Initial
`naf init`

#### Code generate

You can find all possible blueprints in the table below:

Scaffold  | Usage
---       | ---
[Model]      | `naf model`
[Controller, Route, Schema]      | `naf controller`
